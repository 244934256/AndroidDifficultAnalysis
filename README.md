- 换过一块800大洋原装屏幕的【锤子T1】，我支持老罗的情怀，但是这个家伙不让Root，还不能看应用后台详情，这对于我这样的工程师怎么能忍！然后我就很气愤，决定过年回家把手机送给我妈妈看国产电视剧，用锤子看国产电视剧剧应该也算是对老罗的惩罚 ^_^
